impulse :: Int -> [Int32] -> [(Int, Int32)] #

Returns a list impulses

Two adjacent 735 subsamples are compared and if the difference between them passes an (currently arbitrarily chosen by me) threshold it adds the the frame in which it occurs and the magnitude of the difference

>>> impulse counter [DATA.WAVESample] -> [(Int, WAVESample)]
